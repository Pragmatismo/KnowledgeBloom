# Famous Art pack

Contains 20 artworks and 40 learning items: one title question and one artist question per image.

The files in `assets/images/` are labelled placeholders. Replace each while preserving its filename, then update the corresponding `question.credit` entries in `items.json`. Because both questions reference the same file, one replacement updates both.

Stable IDs use `<artwork_id>_title` and `<artwork_id>_artist`.
